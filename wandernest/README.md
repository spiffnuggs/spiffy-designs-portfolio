## 🎯 How to Replace Photos

### Option 1: Using Image URLs
Find the `.story-image` divs and replace the placeholder text with:
```html
<div class="story-image" style="background: url('YOUR-IMAGE-URL.jpg') center/cover;">
</div>
```

### Option 2: Using Local Files
1. Create an `images` folder in the same directory
2. Add your photos there
3. Update the code:
```html
<div class="story-image" style="background: url('images/cottesloe-villa.jpg') center/cover;">
</div>
```
